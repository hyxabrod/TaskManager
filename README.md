# TaskManager

Task manager application built with modern Android technologies.

## Implementation Details

### General

1. **Tech Stack**:
    - Navigation 2
    - Material 3
    - Jetpack Compose
    - Mockk
    - Hilt
    - Room database
2. **Architecture**: Clean Architecture principles.
3. **Persistent storage**

### Main Task Features

1. **Adding**
2. **Deleting**
3. **Creating**
4. **Updating**


### Extra Features

1. **State Preservation**
2. **Theming**: Implemented branded color for the Dark and Light themes 
3. **Task statuses**: TODO, IN PROGRESS, DONE
4. **Filtering**: User can filter by title and statuses
5. **Empty and Error states**
6. **Data generation**
7. **Pagination**

### Space for improvement
1. **Multimodularity**: extract core and features (list and details) into separate modules
2. **Better filtering**: add more sorting options
3. **Templates for tasks**
4. **Backup data to iCloud/Google Drive**
5. **Feedback functionality**
6. **Timer and various notification options for time-limited tasks.**
7. **Adding Images**
8. **Ability to assign priorities**
9. **Filtering by priorities.**
